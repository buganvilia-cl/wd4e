Anquilosaurio

Un disfraz de anquilosaurio para mi hijo.

El proceso

Estructura

Para que fuera redondo usé una pelota, cubierta en cinta de embalar y con "aletas" marcando la mitad. Después de un para de capas de papel maché bien seco, saqué la pelota y uní las mitades con masking tape. Un par de capas más de papel maché y estamos listas para decorar.

Geografía:

Bajé de internet un mapa "en gajos" para armar un globo terráqueo (no guardé el link), ajusté el porte y lo pegué. Luego con papel maché molido, un sitio web con mapas topográficos y mucha paciencia fui formando las principales cadenas montañosas.

Pintura:

Un poco de blanco para borrar las lineas, y pintar con cuidado tratando de copiar los colores de google earth.

Base:

Un palo de anticucho por el eje, y una base simple de cartón para sostenerlo en el ángulo correcto. Puse unos pedazos de tubo plástico donde se pone el eje para que pueda girar.

La base no le hace justicia al globo terráqueo, pero no tuve tiempo de decorarla.
